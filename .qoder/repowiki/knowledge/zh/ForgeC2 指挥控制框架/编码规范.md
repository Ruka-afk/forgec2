- 所有敏感操作（如命令执行、文件删除）均通过 `AuditMiddleware` 记录至 `audit_logs` 表，包含用户、IP 及操作详情。
- Agent 与 Server 间的二进制大对象（如截图、文件块）统一采用 Base64 编码在 JSON Beacon 中传输，并在服务端进行分片组装或流式处理。
- Handler 函数遵循 `handle[Resource][Action]` 命名规范（如 `handleBeacon`, `handleLogin`），并通过 `gin.H` 传递模板数据。
- 配置加载采用 `DefaultConfig` 兜底策略，若 `config.yaml` 不存在则自动创建并填充随机生成的 JWT Secret。
